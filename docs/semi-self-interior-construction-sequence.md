# 반셀프 인테리어 공정 순서 가이드

> 출처: 네이버 블로그 3곳 (맛칠라, 마스터철거, 보담다 목수) 검색 정리
> 기준: 구축 아파트 전체 철거 기준

---

## 전체 흐름

```
준비/행정 → 철거 → 샷시 → 설비(수도) → 방수 → 전기 → 목공
→ 필름 → 도장/퍼티 → 타일 → 도배 → 장판/마루
→ 주방/가구 → 조명·전기마감 → 탄성코트(베란다) → 입주청소
```

---

## 공정별 상세

| 순서 | 공정 | 소요 기간 | 핵심 포인트 |
|------|------|-----------|-------------|
| 0 | **준비/행정** | 공사 전 | 입주민 동의서, 엘리베이터·동선 보양 신청, 확장 시 구청 행위허가 |
| 1 | **철거** | 1~2일 | 살릴 부분/철거 부분 명확히 구분. 욕실 철거도 이때 함께. 장판 아래 마루 등 변수 많음 → 일정 하루 여유 필수 |
| 2 | **샷시** | 1일 | 철거 완료 후 진행. 실측 후 창호 주문까지 최소 1주일 소요. 비 오면 시공 불가(날씨 변수 큼). 샷시 교체 시 목공 단열 공사 필수 |
| 3 | **설비(수도)** | 1~2일 | 배관 위치 변경 시 철거 직후 진행. 이후 바로 방수 |
| 4 | **방수** | 1~2일 | 욕실·세탁실: 설비 → 방수 → 타일 순서 필수. 1차(바닥) → 2차(벽체), 24시간 이상 물막이 테스트 |
| 5 | **전기** | 1~2일 | 콘센트·스위치 위치 미리 확정. 인덕션은 단독 배선 필요. 목공과 일정 긴밀히 조율 |
| 6 | **목공** | 2~3일 | 천장·벽체 구조, TV박스, 문틀, 걸레받이 등. 인테리어 전체 완성도를 좌우하는 핵심 공정 |
| 7 | **필름** | 1일 | 도배 전에 먼저 시공. 문틀·몰딩·문짝 등. 걸레받이 색상 통일 필수 |
| 8 | **퍼티/도장** | 1~2일 | 몰딩 교체 시 퍼티 작업 추가됨(비용 발생 주의). 천장 도장은 도배 전 완료 |
| 9 | **타일** | 3~5일 (양생 포함) | 욕실·주방 타일 같은 팀에 맡기면 비용 절감. 덧방 vs 철거 후 새 시공 선택 |
| 10 | **도배** | 1~2일 | 띄움시공·퍼티 작업 확인 필수. 도배 후 최소 2~3일 건조 |
| 11 | **장판/마루** | 1일 | 도배 다음날 바로 시공 가능 |
| 12 | **주방/싱크대** | 1일 | 바닥 완성 후 설치. 타일 범위 미리 주방팀과 협의 |
| 13 | **조명·전기 마감** | 반~1일 | 콘센트 커버, 스위치, 조명 설치. 누전·점등 테스트 |
| 14 | **탄성코트** | 1일 | 도배 후 최소 2~3일 이후에 진행(건조 방식이 달라서). 베란다 마감 |
| 15 | **셀프 작업** | 수시 | 콘센트·스위치 커버 교체, 조명 교체 등 직접 |
| 16 | **입주청소** | 1일 | 전문 업체 추천 |

---

## 자주 발생하는 변수 & 주의사항

- **철거 추가**: 장판 아래 마루, 썩은 문, 안 쓰는 보일러 버튼 등 열어봐야 나오는 변수가 많음. 철거 일정 뒤에 하루 버퍼 확보
- **샷시 날씨 변수**: 비 오는 날 시공 불가. 실측~시공까지 1주일 여유 필요
- **몰딩 교체 시 퍼티 작업**: 갈매기 몰딩 → 얇은 몰딩으로 바꾸면 퍼티 작업 추가 비용
- **도배 + 탄성코트 간격**: 도배는 문 닫고 말리기, 탄성코트는 창문 열고 건조 → 최소 2~3일 간격 필수
- **타일 욕실+주방 동시**: 같은 팀에 맡기면 비용 절감
- **목공 ↔ 전기 긴밀 협조**: 전기 배선이 목공 공정에 직접 영향
- **철거 견적**: 철거 전 정확한 견적 불가. 열어봐야 자재량·배선·배관 상태 확인 가능
